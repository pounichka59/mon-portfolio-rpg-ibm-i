# Mon Portfolio RPG - IBM i (AS/400)

Ce repository contient mes exercices et mes premiers projets en **RPGLE** sur IBM i.

### Objectif
- Reprendre les bases de l'environnement IBM i
- Progresser en RPGLE (free-format)
- Préparer un portfolio visible pour des recherches d'emploi

### Structure du repository

- **01-bases/**          → Commandes IBM i et concepts de base
- **02-rpg-base/**       → Premiers programmes RPGLE simples
- **03-fichiers/**       → Programmes avec lecture/écriture de fichiers
- **04-sql/**            → SQL embarqué dans RPG
- **05-projets-simples/** → Petits projets complets
- **06-supports/**       → Cours et documents de référence (optionnel)

### Technologies utilisées
- IBM i (pub400)
- RPGLE Free Format
- SQLRPGLE
- (VS Code) + Code for IBM i
